
# Task Management 



## Tech Stack

**Client:** React  T

**Server:** Node, Express

**Database:** Mongodb

    
## Run Locally

Clone the project

```bash
  git clone https://github.com/maran9335/taskmanagement.git
```

Go to the project directory

```bash
  cd client
```

```bash
npm start
```

Start the server

```bash
 server
```
```bash
npm start
```
